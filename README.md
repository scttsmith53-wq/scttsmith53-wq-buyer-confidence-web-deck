## GitHub Desktop steps

1. Unzip this file.
2. Open the unzipped folder.
3. Copy ALL contents into your GitHub repo folder.
4. GitHub Desktop should show changed files.
5. Commit to main.
6. Push origin.

Do not upload:
node_modules
.next
.env.local
